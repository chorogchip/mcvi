# mcvi 아이디어

`mcvi`는 3차원 블록 데이터를 vi 방식으로 편집하는 터미널 편집기이다.

## 핵심

- 데이터는 `x`, `y`, `z` 좌표를 가진 블록 공간으로 다룬다.
- 화면에는 한 번에 하나의 2차원 슬라이스를 보여준다.
- 화면 오른쪽 끝에는 현재 커서의 `(x, z)` 열을 모든 `y` 슬라이스에 대해 압축해서 보여준다.
- 기본 슬라이스는 고정된 `y`의 `x, z` 평면이다.
- vi의 Normal / Insert / Visual / Replace 모드를 블록 편집에 맞게 적용한다.
- 입력 방향을 `+x`, `-x`, `+y`, `-y`, `+z`, `-z` 중 하나로 지정할 수 있다.
- Insert / Visual / Replace 동작은 현재 입력 방향을 기준으로 자라거나 선택된다.
- 블록은 화면에서 알파벳 문자로 표현하고, 실제 블록 이름은 alias로 관리한다.

## 목표

- vi의 조작감을 유지하면서 3차원 블록 편집에 필요한 방향, 슬라이스, 레이어 개념을 추가한다.
- 텍스트 편집 명령을 가능한 한 자연스럽게 블록 편집 명령으로 확장한다.
- 구체적인 키맵과 동작 정의는 `specification.txt`에 둔다.
